### Описание
В приложении реализованы следующие консольные игры:

1. Even - "Это четное число?";
2. Calc - "Какой результат?";
3. GCD - "Чему равен НОД?";
4. Progression - "Какой элемент скрыт?";
5. Prime - "Это простое число?"
